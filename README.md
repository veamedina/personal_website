# Personal Profile Website

This project is a **personal profile website** built using **HTML, CSS, and JavaScript**, showcasing personal information, hobbies, and interactive features.

## Features
- **Responsive layout** for desktop and mobile  
- **Navigation menu** linking to About Me, Hobbies, and Contact sections  
- **Hobby cards** with hover effects and interactive content:  
  - **Reading:** includes favorite books and quotes  
  - **Photography:** displays sample images  
  - **Relaxation & Music:** interactive music player with multiple songs  
    - *"For Always"* by Ed Sheeran  
    - *"Back to Friends"* by Sombr  
- **Random fun fact generator** implemented with JavaScript  
- Interactive **play/pause buttons** for music with dynamic text  

## Technologies Used
- HTML for structure  
- CSS for styling and responsive layout  
- JavaScript for interactivity
